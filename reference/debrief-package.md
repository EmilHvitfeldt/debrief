# debrief: Text-Based Summaries for 'profvis' Profiling Data

Provides text-based summaries and analysis tools for 'profvis' profiling
output. Designed for terminal workflows and AI agent consumption,
offering views including hotspot analysis, call trees, source context,
caller/callee relationships, and memory allocation breakdowns.
